Projet 1 : Gestion de données météorologiques et photovoltaïques

L’objectif était de récupérer les données d’une station météorologique et de deux onduleurs reliés à des panneaux photovoltaïques. Afin de mener à bien ce projet, nous avons utilisé des scripts python pour la récupération des données, un site web en CakePHP pour mettre à disposition ces données aux utilisateur·ice·s et du bash afin de déployer automatiquement notre projet sur un Raspberry PI vierge. A partir de ces données, nous avons utilisé un modèle de deep-learning afin de tenter d’établir des prévisions météorologiques.
